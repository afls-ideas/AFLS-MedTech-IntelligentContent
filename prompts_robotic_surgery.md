# Robotic Surgery — VisuMax eDetailer — Image Generation Prompts

## How to Generate the Slide Images

These prompts are designed to generate 2048 x 1448 pixel images for use as slide backgrounds in an LSC Intelligent Content (CLM) multi-page zip presentation. Each image sits below a 44px HTML navigation bar on a 1024x768 iPad display (Retina 2x).

### Recommended Tools

| Tool | How to Use |
|------|-----------|
| **Google Gemini** | Go to gemini.google.com → Click the image generation icon or just paste the prompt and Gemini will detect it as an image request. Download the result as PNG. |
| **ChatGPT (DALL-E)** | Go to chatgpt.com → Paste the prompt. ChatGPT will generate the image via DALL-E. Download as PNG. |
| **Midjourney** | Paste the prompt into the /imagine command in Discord or the Midjourney web app. Append `--ar 256:181` to maintain the 2048x1448 aspect ratio. Upscale the result. |
| **Adobe Firefly** | Go to firefly.adobe.com → Text to Image → Paste the prompt. Set aspect ratio to Custom and enter 2048x1448. |

### Tips

- If the tool doesn't support exact pixel dimensions, use an aspect ratio of approximately **1.41:1** (close to 256:181).
- After generation, resize/crop to exactly **2048 x 1448 pixels** for pixel-perfect fit.
- Save as PNG for best quality. JPEG is acceptable but may introduce artifacts on text.
- Place generated images in the `assets/` folder with the naming convention `01_Intro.png`, `02_Features.png`, etc.
- If a tool produces text that is misspelled or garbled, regenerate or use an image editor to fix the text overlays manually.
